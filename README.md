# Portfolio Public

Nueva versión modular y escalable del portafolio profesional, basada en Next.js 16+, Feature-Sliced Design y buenas prácticas modernas.

## Estructura
- Modular, profesional y lista para crecer
- Home completo como base
- Arquitectura lista para nuevas features y páginas

## Scripts
- `npm run dev` — desarrollo local
- `npm run build` — build producción
- `npm run start` — server producción

## Stack
- Next.js 16+
- React 18+
- TailwindCSS
- TypeScript
